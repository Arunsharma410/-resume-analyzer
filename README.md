# 🚀 AI Resume Analyzer

> Production-quality MERN stack application that uses Google Gemini AI to analyze resumes against job descriptions.

## ✨ Features

- 🔐 JWT Authentication
- 📄 PDF Resume Upload & Parsing
- 🤖 AI-Powered Analysis (Google Gemini)
- 📊 Match Score & ATS Compatibility
- 🎯 Skills Gap Analysis
- 💼 Job Recommendations
- 📈 Analysis History
- 👤 User Profile Management
- 🎨 Beautiful Dark Theme UI
- 📱 Fully Responsive

## 🛠️ Tech Stack

**Frontend:**
- React 18 + Vite
- Tailwind CSS
- React Router v6
- Recharts
- Lucide Icons

**Backend:**
- Node.js + Express
- MongoDB + Mongoose
- JWT Authentication
- Multer + pdf-parse
- Google Gemini AI
