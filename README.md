# 🚀 AI Resume Analyzer

> Production-quality MERN stack application that uses Google Gemini AI to analyze resumes against job descriptions.

## ✨ Features

- 🔐 JWT Authentication
- 📄 PDF Resume Upload & Parsing
- 🤖 AI-Powered Analysis (Google Gemini)
- 📊 Match Score & ATS Compatibility
- 🎯 Skills Gap Analysis
- 💼 Job Recommendations
- 📈 Analysis History
- 👤 User Profile Management
- 🎨 Beautiful Dark Theme UI
- 📱 Fully Responsive

## 🛠️ Tech Stack

**Frontend:**
- React 18 + Vite
- Tailwind CSS
- React Router v6
- Recharts
- Lucide Icons

**Backend:**
- Node.js + Express
- MongoDB + Mongoose
- JWT Authentication
- Multer + pdf-parse
- Google Gemini AI

## 📦 Installation

### Backend
\`\`\`bash
cd backend
npm install
npm run dev
\`\`\`

### Frontend
\`\`\`bash
cd frontend
npm install
npm run dev
\`\`\`

## 🔑 Environment Variables

### Backend (.env)
\`\`\`
MONGODB_URI=your_mongodb_uri
JWT_SECRET=your_jwt_secret
GEMINI_API_KEY=your_gemini_key
FRONTEND_URL=http://localhost:5173
\`\`\`

### Frontend (.env)
\`\`\`
VITE_API_URL=http://localhost:5000/api
\`\`\`
